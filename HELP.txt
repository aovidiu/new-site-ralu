2. cd contact-worker
3. npx wrangler dev (startez cloudefare local) care va asculta pe  http://127.0.0.1:8787
4. in contact-email-componet.ts modific const url = 'http://127.0.0.1:8787/contact';

2. npx wrangler secret put FORMSPREE_URL
3. apoi la prompt fac paste cu noul URL
4. npm run deploy (deploy la noul config pe serverul cloudfare)
